以下配置，数据库使用 mariadb5.5.6 版本

配置 django 环境

C:\Users\FanMing\Desktop>virtualenv  djanodemo22

C:\Users\FanMing\Desktop\djanodemo22\Scripts>activate

(djanodemo22) C:\Users\FanMing\Desktop\djanodemo22\Scripts>pip install django

(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>pip install mysqlclient

(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>python manage.py makemigrations        #生成迁移的文件
(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>python manage.py migrate            #根据文件生成表
(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>python manage.py createsuperuser

配置使用数据库

#myapp/models.py

from django.db import models

class Question(models.Model):
    question_text = models.CharField(max_length=200,default='python')
    pub_date = models.DateTimeField(auto_now=True,name='上传时间')

#myapp/admin.py

from django.contrib import admin


from .models import Question

admin.site.register(Question)

#setting.py 注册应用

INSTALLED_APPS = [
    ...
    'myapp',        #添加应用
]

...

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',   #使用MySQL数据库
        'NAME': 'djangoDemo',       #数据库名字
        'USER': 'user1',            #用户名
        'PASSWORD': '123',          #密码
        'HOST': '192.168.3.128',    #数据库地址
        'PORT': '3306',             #数据库端口
    }
}

根据 models.py 生成表

(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>python manage.py makemigrations     #生成迁移的文件
(djanodemo22) C:\Users\FanMing\Desktop\djangoDemo22>python manage.py migrate            #根据文件生成表
